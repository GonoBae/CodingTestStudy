## 목표

- 코딩테스트 대비


## 계획
- Level 1

## 규칙

- 매일 한 문제씩 풀기
- 수요일 + 일요일 코드리뷰
- 푼 문제
  - README 표에 :heavy_check_mark: 표시 하고 이모지에 본인 풀이 링크 달 것.

- 못 푼 문제
  - README 표에 :question: 표시
  - 다른 사람의 코드를 참고하여 풀었을 경우, 해당 풀이를 블로그에 작성하고 이모지에 링크 달 것.



## 사용 이모지

- 아직 안 푼 문제 : :heavy_multiplication_x:
- 모르는 문제 : :question:
- 푼 문제 : :heavy_check_mark:
