- 수요일 + 일요일 코드리뷰
- 푼 문제

- 못 푼 문제
  - README 표에 :question: 표시
- 다시 볼 문제
  - README 표에 :pencil: 표시
  - 다른 사람의 코드를 참고하여 풀었을 경우, 주석 꼼꼼하게 달고 `본인 풀이 링크 달 것.`
